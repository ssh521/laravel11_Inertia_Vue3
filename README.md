# laravel11_inertia_vue3

github: https://github.com/ssh521/laravel11_inertia_vue3.git

## Requirements
- Laravel 11
- NodeJS 20
- Composer
- NPM
- TailwindCSS

## Setup
`cp .env.example to .env`

`php artisan key:generate`

`composer install`

`php artisan migrate`

`npm install`

`npm run dev`

`php artisan serve`
