# laravel11_inertia_vue3

github: https://github.com/ssh521/laravel11_inertia_vue3.git

## Requirements
- Laravel 11
- NodeJS 20
- Composer
- NPM
- TailwindCSS

## Setup
```
composer install

npm install

cp .env.example .env

php artisan key:generate

php artisan migrate

npm run dev

php artisan serve
```
