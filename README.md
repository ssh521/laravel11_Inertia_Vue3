# laravel11_Inertia_vue3

Inertia + Vue 3 + TailwindCSS + Laravel 11 기본 프로젝트

github: https://github.com/ssh521/laravel11_inertia_vue3.git

## Requirements
- Laravel 11
- Vue 3
- Inertia
- TailwindCSS

## Setup
```
composer install

npm install

cp .env.example .env

php artisan key:generate

database.sqlite 파일 생성

php artisan migrate

npm run dev

php artisan serve
```
